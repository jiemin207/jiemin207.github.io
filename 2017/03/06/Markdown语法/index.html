Markdown语法

发表于 2017-03-06

前言

本博客采用的是markdown语法书写，所以打算先学习下markdown语法，顺便记录一下，加深一下印象。

Markdown语法规则

标题

在Markdown中,如果一段文字被定义为标题，只要在这段文字前加#号就行。

# 一级标题

# 二级标题

# 三级标题

以此类推，总共六级标题，需要注意的是在#与文字之间需要有空格。

列表

列表分为有序列表和无序列表。在Markdown中，只要在文字前加-或者*就可变成无序列表。而有序列表则直接在文字前加1. 2. 3.符号和文字之间加上一个字符的空格

- 第一

- 第二

- 第三

效果

  • 第一
  • 第二
  • 第三

1. 第一

2. 第二

3. 第三

效果

  1. 第一
  2. 第二
  3. 第三

引用

如果你需要引用一段别处的句子，那么就要用引用格式。引用格式只需要在文本前加入>这种尖括号(大于号)即可

> 这是段引用

效果

这是段引用
