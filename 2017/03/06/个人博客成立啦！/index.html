个人博客成立啦！

发表于 2017-03-06

成立背景

由于买了个搬瓦工的vps，我在上面搭建了openvpn和shadowsocks后，发现还有不少剩余空间，于是想在上面再做些什么。在网上查了点资料后决定在vps上搭建一个个人博客。这个个人博客采用Hexo框架搭建，使用了next的主题风格。

用途

  1. 纪录学习笔记
  2. 收藏技术文档
  3. 发布技术心得
  4. 转载心灵鸡汤
  5. 写写自己感兴趣的事

目标

每个星期至少写一篇原创的技术文章，可以是自己的学习笔记，也可以是自己在项目中碰到的问题，以及解决办法。

期望

我自去年转行成程序员以后，便想把我所学的知识再好好梳理一遍，把今后要学的东西找个地方纪录下来，而博客就是个很好的工具。对于博客之类的东西，我曾有过新浪博客、QQ空间等，但写过几篇文章之后就被我扔在一边了，再也没去打理过，希望这次我可以坚持下去！
